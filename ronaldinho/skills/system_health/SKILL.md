# Skill: System Health Check

## Objective

Monitor and report the health of the host system.

## Capabilities

1. **CPU/Memory Usage**: Check load levels using `top` or `free`.
2. **Disk Space**: Monitor disk usage with `df -h`.
3. **Service Status**: Check if background processes are running.

## Instructions

Quando o usuário perguntar "Como está o sistema?", use o terminal para coletar estatísticas de CPU, memória e disco e apresente um resumo fenomenal.
